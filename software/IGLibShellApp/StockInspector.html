About the Software

  In summer 2017, it was clear that the next large financial crisis is approaching. I have estimated that there is 70% chance that financial crisis will begin before Summer 2018, and more than 90% chance that it will begin before Summer 2019. I needed some tools that would help me predict whether the time when the crisis would strike is approaching. Even more, I was wanted to build some tools that would tell me when the stock markets would reach the bottom at the next end of financial crisis. W did not count on forecasting this much in advance, but it would help a lot if I could estimate with higher probability whether in the given moment the bottom is already reached or not.

  Related to this, an idea was born that I would create a simple graphical tool that would enable easy inspection of behavior of stock markets, with some instructional graphical representations that would help get the feeling about where the markets currently are and what are the chances that trends would change significantly. I named this software Stock Inspector.

Other Software 

Stock Inspector is based on the Investigative Generic Library (IGLib.NET).
Some other publically available software based on this library includes:
  •  NeurApp – an educational application for visually exploring features of function approximation with artificial neural networks (ANN). It creates 1D and 2D ANN models of user defined functions and provides visualization capabilities to compare these models with originals. It can be downloaded from Softpedia.
  •  AnnApp – software for exploring multidimensional ANN-based models.
  • IGShell - a shell application based on IGLib.
  • Oscillator Modeller – an educational application for playing with nonlinear driven oscillators and their resonance curves.
