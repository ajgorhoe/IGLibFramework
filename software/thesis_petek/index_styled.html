About the Software

This page contains an application for modelling behavior of linear and nonlinear oscillators, together with the Investigative Generic Library (IGLib) binaries.

The application was created by Marko Petek in the scope of his graduate thesis. It features a general 4th order Runge Kutta solver for systems of ordinary differential equations, a number of special solver classes for solution of equations that drive motion of oscillators, a command-line interpreter acting as driver of the application and user interface through which one can define what the application should do, and a flexible object oriented scripting engine that enables the users to extend the existent functionality of the software. IGLib was used as code base and application framework on which the application was built. The application is written entirely in C# and is compiled for 64 bit .NET platforms.

Download

Download the files (click on the link).

Short installation instructions:

  • Download the compressed archive by clicking on the download link above.
  • Extract contents of the archive to an arbitrary directory.
  • Run the executable from the location where files were unpacked. As executable is build around a command-line interpreter, things are invoked by providing commands and their arguments as arguments to the program that is run in a console. Some examples of commands with arguments are in the .cmd file that is contained in the root directory.
  • Within the root directory, there is a Visual Studio C# project that can be used to build the application. You can modify the application and run it from the VS IDE.
