About the SendIgence

 
  SendIgence is a software for efficient management of periodic or bulk e-mail sending.  It is simple to use and it can be run either interactively or by using configuration files in XML format. Configuration files are usually prepared in interactive SendIgence session or edited manually (an XML editor is recommended).

  The software incorporates an easy-to-use but powerful set of utilities for sending large amount of e-mails, dividing recipients to smaller groups, timing the process of sending e-mails, extracting addresses from files, etc.

Features:
  • Use of different command line e-mail clients (although sendEmail is recommended).
  • Sending mails to large number of addresses.
  • Dividing the whole set of addresses to smaller subgroups.
  • Defining pause between sending the mail to subgroups of recipients.
  • Storing sessions to XML files.
  • Checking the e-mails before sending to final recipients.
  • Extracting addresses from files.
  • Extracting addresses from multiple files (ability to browse directories recursively).
  • Eliminating repeating addresses.
  • Defining exclusion lists.
  • Distributing internet traffic more evenly by sending e-mails over larger periods of time.
  • Choosing the number of recipients and pause between messages randomly (with specified average and maximal deviation).
  • Address book in XML.
  • Address center with LDIF, XML and arbitrary text files containing e-mail addresses.

The program is run from a terminal window and it is simple to use. Although it is not stand-alone and needs another e-mail client, this is not a major obstacle. The free command line e-mail client sendEmail is included in the program's distribution packages. This e-mail client is used by default, however the program can be configured to utilize other e-mail clients that can be run form command-line.

Download the program

 
  Currently, only a MS Windows version is available. I hope somebody would provide the necessary assistance in order to create ports to other platforms. At least I would like to provide a version for Linux.

  The latest MS Windows distribution is available as a ZIP archive. Download the file, unzip it to a chosen directory (it doesn't matter which one) and copy all executables (actuallty the whole contents of the bin\ subdirectory) into a directory that is included in the system's path. Then just execute the program (e.g. by clicking on its from the Explorer or by typing its name in the command prompt) and enjoy exploring its capabilities. You will find all necessary information in the documentation.
